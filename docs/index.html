Retro CNC Panel

A 1980s terminal-styled web interface for controlling 3D printers and CNC machines running Klipper/Moonraker

QUICK INSTALL

curl -sSL https://justinh-rahb.github.io/moonraker-cnc/install.sh | bash

Installs to ~/retro-cnc-panel with optional nginx configuration. Run the same command to update.

Real-Time Control

WebSocket-powered live position tracking, multi-axis jogging, Z-offset baby-stepping, and instant homing controls.

Print Management

Live progress, speed, and flow metrics. File browser with uploads. Pause/Resume/Cancel controls.

Temperature Monitoring

Auto-discovers all sensors with live graph, customizable presets, and multi-extruder support.

G-code Console

Send commands directly to Klipper with command history and real-time response logging.

Machine Limits

Configure velocity, acceleration, and corner velocity. Auto-detects Klipper version.

Custom Macro Panels

Create unlimited macro panels with customizable buttons, labels, and colors.

Extruder Control

Flow rate, pressure advance, smooth time tuning, and filament load/unload macros.

Retro Aesthetic

CRT scanlines, glowing displays, color-coded LEDs, and tactile button feedback.
