Retro CNC Panel

A 1980s terminal-styled web interface for controlling 3D printers and CNC machines running Klipper/Moonraker

QUICK INSTALL

curl -sSL https://raw.githubusercontent.com/justinh-rahb/moonraker-cnc/main/docs/install.sh | bash

Installs to ~/retro-cnc-panel with optional nginx configuration. Run the same command to update.

Real-Time Control

WebSocket-powered live position tracking, multi-axis jogging, and instant homing controls.

Temperature Monitoring

Auto-discovers all temperature sensors with live graph visualization.

Custom Macros

8 configurable macro buttons for your most-used G-code commands.

Retro Aesthetic

CRT scanlines, glowing displays, and tactile button feedback.
